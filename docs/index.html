üíª Usage Examples

Basic Usage

from easy_slam import EasySLAM # Initialize with webcam and ORB-SLAM slam = EasySLAM(sensor='webcam', algorithm='orb_slam') # Start processing slam.start() # Stop when done slam.stop()

Advanced Usage with GUI

# Launch the GUI easy-slam-gui # Or use CLI with advanced features easy-slam --sensor realsense --algorithm rgbd_slam --enable-semantic --enable-fusion

Python API

# Custom configuration config = { 'sensor': 'realsense', 'algorithm': 'rgbd_slam', 'enable_semantic': True, 'enable_sensor_fusion': True, 'output_dir': 'results/' } slam = EasySLAM(**config) slam.start()

Dependencies

EasySLAM automatically installs all required dependencies including OpenCV, NumPy, PyQt6, and more. For advanced features, additional packages like PyRealSense2 and YOLOv8 are installed as needed.
